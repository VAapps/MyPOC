#### DOS command to delete bin & obj Folders
```
@echo off
@echo Deleting all BIN and OBJ folders...
for /d /r . %%d in (bin,obj) do @if exist "%%d" rd /s/q "%%d"
@echo folders have been deleted. Press any key to exit
pause > nul
```
